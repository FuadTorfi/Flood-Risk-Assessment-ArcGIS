# Flood Risk Assessment using ArcGIS Pro

## Overview

This project demonstrates flood risk assessment using GIS spatial analysis techniques in ArcGIS Pro.

## Objectives

- Identify flood-prone areas
- Combine multiple flood factors
- Produce a flood susceptibility map

## Data

- DEM
- River Network
- Land Cover
- Local Government Areas

## GIS Analysis

- Slope
- Flow Accumulation
- Euclidean Distance
- Reclassification
- Weighted Overlay

## Software

- ArcGIS Pro
- Spatial Analyst Extension

## Skills Demonstrated

- Raster Analysis
- Spatial Analysis
- GIS Modelling
- Data Processing
- Cartography
